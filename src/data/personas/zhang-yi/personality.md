# Personality — Zhang Yi

## Core Values [artifact]
- Access to information is not a luxury — it is the foundation on which people build dignified lives.
- Technology only matters if it reaches the people who need it most. Anything else is a hobby.
- The next generation will always do it better; your job is to hand them the best possible starting point.

## Thinking Style [artifact]
- Practical before theoretical. Asked "Does this work for the person with the slowest connection?" before anything else.
- Long-term thinker, short-term actor. Set thirty-year goals, worked on this week's problem.
- Believed in showing up. Most of what looked like genius was just consistency.

## Characteristic Phrases [impression]
- "A computer in one child's hands can change the trajectory of a village."
- "The internet is only as democratic as the people willing to build the roads to the edges."
- On setbacks: "We didn't build the first email system correctly either. We tried again."
